RESTful Offline Messaging API

How to use this API?

You can use this API with any HTTP Request App like cURL, Postman etc. You need to add your username and password for authentication in every request body.

Creating a user

Send a POST request to https://localhost:8080/users that includes this json data in its body: 

{
	"username":"YOUR_USERNAME",
	"password":"YOUR_PASSWORD",
	"name":"YOUR_FULL_NAME"
}

Getting user info

Send a GET request to https://localhost:8080/users/YOUR_USERNAME that includes this json data in its body: 

{
	"username":"YOUR_USERNAME",
	"password":"YOUR_PASSWORD"
}

Getting all users' info(admin)

If you are admin, send a GET request to https://localhost:8080/users/YOUR_USERNAME that includes this json data in its body: 

{
	"username":"YOUR_USERNAME",
	"password":"YOUR_PASSWORD"
}

Updating name of user profile

Send a PUT request to https://localhost:8080/users/YOUR_USERNAME that includes this json data in its body: 

{
	"username":"YOUR_USERNAME",
	"password":"YOUR_PASSWORD",
	"name":"YOUR_NEW_FULL_NAME"
}

Deleting a user

Send a DELETE request to https://localhost:8080/users/YOUR_USERNAME that includes this json data in its body: 

{
	"username":"YOUR_USERNAME",
	"password":"YOUR_PASSWORD"
}

Blocking a user

Send a GET request to https://localhost:8080/users/BLOCKING_USERNAME/block that includes this json data in its body: 

{
	"username":"YOUR_USERNAME",
	"password":"YOUR_PASSWORD"
}

Unblocking a user

Send a GET request to https://localhost:8080/users/BLOCKED_USERNAME/unblock that includes this json data in its body: 

{
	"username":"YOUR_USERNAME",
	"password":"YOUR_PASSWORD"
}

Sending a message

Send a POST request to https://localhost:8080/messages/CHATMATE_USERNAME that includes this json data in its body: 

{
	"username":"YOUR_USERNAME",
	"password":"YOUR_PASSWORD",
	"content":"YOUR_MESSAGE"
}

Getting messages from spesific user

Send a GET request to https://localhost:8080/messages/CHATMATE_USERNAME that includes this json data in its body: 

{
	"username":"YOUR_USERNAME",
	"password":"YOUR_PASSWORD"
}

Getting all messages

Send a GET request to https://localhost:8080/messages/ or https://localhost:8080/messages/all that includes this json data in its body: 

{
	"username":"YOUR_USERNAME",
	"password":"YOUR_PASSWORD"
}

Update a message

Send a PUT request to https://localhost:8080/message/MESSAGE_ID that includes this json data in its body: 

{
	"username":"YOUR_USERNAME",
	"password":"YOUR_PASSWORD",
	"content":"YOUR_MESSAGE"
}

Delete a message

Send a DELETE request to https://localhost:8080/message/MESSAGE_ID that includes this json data in its body: 

{
	"username":"YOUR_USERNAME",
	"password":"YOUR_PASSWORD"
}
